Python+Scrapy 爬取一亩三分地录取信息

爬虫的小应用-爬取一亩三分地录取信息

作为一个苦逼的出国党，在申请时，定位莫过于坠痛苦的事情，特别是在某石油高中大学, 你根本找不到自己DIY出国的人， 所以拿往届学长学姐定位是不可能的，因此，为了帮助自己定位，我开发了这个小工具， 他能帮助你获取一亩三分地上的所有录取信息，包括申请人的本科学校，专业，GPA，Tofel，GRE，录取学校，录取专业，录取学位等等，希望能帮助到你，也能帮助我自己

进度

  • 目前已经完成
    • 数据的爬取模块，并生成JSON文件
    • 写到数据库里去
  • TODO:
    • 实现定时爬取模块，并挂到我正在写的网站上去
    • 实现一个在线定位系统
