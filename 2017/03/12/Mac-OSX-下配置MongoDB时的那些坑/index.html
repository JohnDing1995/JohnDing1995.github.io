Mac OSX 下配置MongoDb时的那些坑

准备装个MongoDb来存储爬到的数据，结果发现虽说可以通过
brew services start mongodb
来启动MongoDb。但是，当我按照教程输入mongod时，却始终链接不上数据库
如下图所示

可见，是/data/db没有写入权限的问题
一开始，我用了各种更改权限的方法 ，都无济于事。之后通过查阅starkoverflow 发现，只要加sudo，就可完美解决问题。
这个经历告诉我
